# 框架配置目录
> ## config.php     主配置文件
> ### 配置项：
> * *error_reporting*       错误抑制
> * *session_start*         session开关

> ## db.php         数据库配置文件
> ### 配置项：
> * *user*                  用户名
> * *pass*                  密码
> * *db*                    数据库类型，支持PDO数据库类型
> * *host*                  数据库地址，域名或ip
> * *db_name*               数据库名
> * *charset*               字符集类型

> ## waf.php        WAFPHP框架配置文件 *搁置*
